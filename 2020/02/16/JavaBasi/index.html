总结
  1. (1) Java语言规范：定义Java语法
    (2) API(Application Program Interface,应用程序接口)：定义Java库
    (3) JDK(Java Development Toolkit,Java开发工具包)：用于编译、运行和测试Java程序的软件
    (4) JRE(Java Runtime Environment)：运行Java程序的程序
    (5) IDE(Integrated Development Environment,集成开发环境)：快速开发Java程序
  2. 源程序是高级语言编写的程序
  3. 计算机是存储和处理数据的电子设备，包括硬件和软件两部分
