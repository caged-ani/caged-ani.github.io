Java基本程序设计笔记（一）

算法可以帮助程序员在使用程序设计语言编写程序之前进行规划
算法可以用自然语言或者伪代码（自然语言和程序设计语言混在一起使用）描述。
例如计算员面积的算法描述如下：
(1)读入半径
(2)利用公式计算面积
(3)显示面积

编写代码的过程实际是在将算法翻译成程序

  1. 为了存储半径值要先声明变量

  2. 每个变量都有名字、类型、大小和值

  3. 列出程序中对应的每行语句执行后的变量叫做跟踪一个程序，有助于理解一个程序是如何工作的，也是查找程序错误的非常有用的工具
    改进程序，使之可以从键盘读取半径值

    public class ComputeAreaWithConsoleInput {

 public static void main(String[] args) {

     //创建一个Scanner类
     Scanner input = new Scanner(System.in);

     //提示用户输入半径
     System.out.print("Enter a number for radius: ");
     double radius = input.nextDouble();//1. 调用方法读取半径

     //2. 代入公式，计算面积
     double area = radius * radius * 3.14159;

     //3. 显示面积
     System.out.println("The area for the circle of radius "+radius+" is "+area);

 }

}
