6.(倒置数组) 将参数中传递的数组倒置，并返回该数组

public static int gcd(int... numbers)

9.(冒泡排序) 冒泡排序算法遍历数组几次。在每次遍历中，对相邻的两个元素进行比较。如果这一对元素是降序，则交换它们的值；否则，保持不变。由于较小的值像气泡一样逐渐”浮向顶部”，同时较大的”沉向”底部，所以，这种技术称为冒泡排序法(bubble sort)或下沉排序法(sinking sort)。编写一个测试程序，读取10个double型值，调用这个方法，然后显示排好序的数字

10.(修改选择排序) 重复地在当前数组中找到最大值，然后将这个最大值与数组中的最后一个数进行交换。编写一个测试程序，读取10个double型值，调用该方法，并显示排好序的数字

11.(计算一个字符串中大写字母的数目) 编写程序，从命令行输入一个字符串，然后显示字符串中大写字母的数目

12.(仿真：优惠券收集问题) 优惠券收集问题是一个经典的统计问题，它有很多实际应用。这个问题是重复地从一组对象中拿出一个对象，然后求出将所有对象都至少拿出来一次，需要拿多少次。该问题的一个变体是，从一副打乱的52张牌中重复选牌，直到每种花色都选过一张，需要选多少次。假设在选下一张牌之前，将选出来的牌放回去。编写程序，模拟要得到四张不同花色的牌所需要的的选取次数，然后显示选中的四张牌(有可能一张牌被选了两次)

13.(完全相同的数组) 如果list1和list2对应的元素都相等，那么认为list1和list2是完全相同的。使用下面的方法头编写一个方法，如果list1和list2完全相同，则返回true：

public static boolean equals(int[] list1, int[] list2)

14.(数学：组合) 编写一个程序，提示用户输入10个数，然后显示从这10个数中选出两个数的所有组合

15.(模式识别：四个连续相等的数) 编写下面的方法，测试某个数组是否有四个连续相同值的数字

public static boolean isConsecutiveFour(int[] values)

提示用户输入一个整数列表，如果这个列表中有四个连续且具有相同值的数，那就显示该结论。程序应该首先提示用户键入输入的大小，即列表中值的个数

16.(列表分区) 编写以下方法，使用第一个元素对列表进行分区，该元素称为中心点(pivot)。(存疑)

public static int partition(int[] list)

分区后，列表中的元素被重新安排，在中心点元素之前的元素都小于或者等于该元素，而之后的元素都大于该元素。方法返回中心点元素位于新列表中的下标。例如，假设列表是{5,2,9,3,8}，分区后，列表变为{3,2,5,9,8}。最多进行list.length次比较来实现该方法。编写一个测试程序，提示用户输入一个列表的大小以及内容，然后显示分区后的列表

17.(对字符串中的字符排序) 使用下面的方法头编写一个方法，返回一个排好序的字符串

public static String sort(String s)

例如，sort("acb")返回abc。编写测试程序，提示用户输入一个字符串，显示排好序的字符串

18.(游戏：八皇后问题) 经典的八皇后难题是要将八个皇后放在棋盘上，任何两个皇后都不能互相攻击(即没有两个皇后实在同一行、同一列或者同一对角上)。可能的解决方案有很多。编写程序显示一个这样的解决方案
