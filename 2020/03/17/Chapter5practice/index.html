4.(找出得分最高的学生) 提示用户输入学生的个数、每个学生的名字及其分数，最后显示得最高分的学生的名字。使用next()方法而不是nextLine()方法来读取名字

6.(求满足 $n^2>12000的n$ 的最小值) 使用while循环找出满足 $n^2>12000$ 的最小整数n

7.(计算最大公约数) 求两个整数n1和n2的最大公约数可以：首先找出n1和n2的最小值d，然后依次检验d，d-1，d-2，…，2,1是否是n1和n2的公约数。第一个满足条件的公约数就是n1和n2的最大公约数。编写程序，提示用户输入两个正整数，然后显示最大公约数

14.(显示每月第一天是星期几) 编写程序，提示用户输入年份和代表该年第一天是星期几的数字，然后在控制台上显示该年各个月份的第一天是星期几。

15.(金融应用：复利值) 假设你每月在储蓄账户上存100美元，年利率是5%。那么每月利率是0.05/12 = 0.00417。在第一个月后账户上变成：

100 * (1 + 0.00417) = 100.417

第二个月之后账户上变成：

(100 + 100.417) * (1 + 0.00417) = 201.252

第三个月之后账户上变成：

(100 + 201.252) * (1 + 0.00417) = 302.507

以此类推。

编写程序提示用户输入一个金额数(例如：100)、年利率(例如：5)以及月份数(例如：6)，然后显示给定月份后账户上的钱数

17.(游戏：剪刀石头布) 程序提示用户随机产生一个数，这个数为0、1、或2，分别表示剪刀、石头和布。提示用户输入值0、1或2，然后显示一条信息，表示用户和计算机谁赢了游戏，或打成平手，直到用户或计算机应对手两次以上为止一直玩下去
