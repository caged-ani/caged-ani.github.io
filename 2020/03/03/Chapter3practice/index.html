Java学习练习题(二)

Posted on 2020-03-03 Edited on 2020-03-05
基于Java学习笔记(二)的17道练习题

1.(游戏：三数相加)程序产生三个随机数，并提示用户输入这三个随机数的和

2.(随机月份)编写一个随机产生1和12之间整数的程序，并根据数字1,2，……，12显示对应的英文月份

3.(健康应用：BMI)让用户输入重量、英尺和英寸。注意：1英尺 = 0.3048米

4.(对三个整数排序)编写程序，提示用户输入三个整数。以非降序的形式显示这三个整数

5.(游戏：减法测试)随机产生一个计算两个小于100的整数的减法问题

6.(回文数字)提示用户输入一个三位的整数，然后确定它是否是回文数。如果一个数字从左到右以及从右到左都是一样的，这个数字称为回文数。负数的处理和正数一样

7.(游戏：猜硬币的正反面)随机产生一个正数0或1，它们分别代表硬币的正面和反面。程序提示用户输入一个猜测值，然后报告这个猜测值是正确还是错误的

8.(随机点)显示矩形中一个随机点的坐标。矩形中心位于(0, 0)、宽100，高200

9.(运输成本)一个公司使用下面的函数，根据运输重量(以磅为单位)来计算运输成本(以美元计算)

编写一个程序，提示用户输入包裹重量，显示运输成本。如果重量大于20，显示信息the package can not be shipped。如果重量等于或小于0，则显示信息Invalid input

10.(科学：风寒温度)在华氏 $-58^\circ$ 到 $41^\circ$ 之间，并且风速大于2的情况下风寒的温度公式表示为

$t_a$ 是室外温度，$v$ 是速度。编写程序，提示用户输入一个温度值和一个风速值。如果输入值是合法的，那么显示风寒温度，否则显示一条消息，表明温度或风速是不合法数值

11.(几何：点是否在圆内？)提示用户输入一个点(x, y)，然后检查这个点是否在以原点(0, 0)为圆心、半径为10的圆内

12.(游戏：抽牌)模拟从一副52张的牌中抽一张牌。程序显示牌的大小(Ace、2、3、4、5、6、7、8、9、10、Jack、Queen、King)以及牌的花色(Clubs(黑梅花)、Diamonds(红方块)、Hearts(红心)、Spades(黑桃))

13.(使用操作符 && 、||和 ^)提示用户输入一个整数值，然后判断它能否被5和6整除，能否被5或6整除但是不能同时被它们整除

14.(几何：两个矩形)提示用户输入两个矩形中心点的 $x$ 坐标和 $y$ 坐标以及矩形的宽度和高度，然后判断第二个矩形是在第一个矩形内，还是和第一个矩形重叠

15.(当前时间)提示用户输入相对于GMT的时区偏移量，然后以12小时时钟显示这个特定时区的时间

16.(几何：点的位置)给定一个从点 $p0(x0, y0)$ 到 $p1(x1, y1)$ 的有向线段，可以使用下面的条件来确定定点 $p2(x2, y2)$ 是在线段的左侧、右侧，或者在该线段上

编写一个程序，提示用户输入三个点 $p0、p1和p2$ ，显示 $p2$ 是在从 $p0$ 到 $p1$ 的线段左侧、右侧还是在线段上

17.(几何：线段上的点)上题显示了如何测试一个点是否在一个无限长的直线上。修改程序测试一个点是否在一个线段上。提示用户输入三个点 $p0、p1和p2$ ，显示 $p2$ 是否在从 $p0$ 到 $p1$ 的线段上
