Java学习笔记(二)

Posted on 2020-03-02
选择：if语句常见的错误和陷阱、逻辑操作符、switch语句、 条件操作符等

1.Java有六种选择语句：单分支 if 语句、双分支 if-else 语句、嵌套 if 语句、多分支 if-else 语句、switch 语句和条件操作符
2.嵌套 if-else 语句对嵌套深度没有限制，但是使用多分支 if-else 语句可以避免深度缩进，试程序易读

(5)两个浮点数值的相等测试

4.使用 Math.random() 来获得一个0.0到0.1之间的随机 double 值
5.定义常量可以使程序易于阅读
6.对所有程序都应该先编写少量代码然后进行测试，之后再继续添加更多的代码。这个过程称为递进式开发和测试(incremental development and testing)。这种方法能让调试变得更容易，因为你的错误很可能就在你刚刚加进去的代码中
7.操作符 ^ 异或，p1^p2等同于p1 != p2
8.德摩根定律
!(condition1 && condition2)等同于!condition1 || !condition2
!(condition1 || condition2)等同于!condition1 && !condition2
&& 和 ||被称为短路或条件操作符；AND(&)和OR(|被称为无条件操作符)
9.switch表达式必须能计算出一个 char、byte、short、int、或者String型值
10.Java中唯一的三元(条件)操作符：? 和 : 一起出现
boolean-expression ? expression1 : expression2
11.所有二元操作符都是左结合的，只有赋值操作符是右结合
12.调试(debugging)的方法：
(1)手工跟踪(hand trace)或插入打印语句:适用于短小、简单的程序
(2)使用调试工具:适用于庞大、复杂的程序
13.一些调试工具的有用特征：
·一次执行一条语句
·跟踪进入或者一步运行一个方法
·设置断点
·显示变量
·显示调用堆栈
·修改变量
