循环

1 循环分类
按结构分：while、do-while、for循环
按计数方式分：计数器控制的循环、标记位控制的循环

3 输入、输出重定向
在需要输入大量数据的情况下，可以将这些数据用空格隔开，保存在一个名为 input.txt 的文本文件中，然后使用下面这个命令运行这个程序：

java ClassName < input.txt // 输入重定向

或者将输出发送给文件：

java ClassName > output.txt // 输出重定向

可同时使用：

java ClassName < input.txt > output.txt

4 循环的选择
while和for是前测循环(pretest loop)，do-while是后测循环(posttest loop)

如果已知重复次数，采用for循环；

无法确定重复次数，采用while循环；

如果在检测继续条件前需要执行循环体，则采用do-while循环。

5 最小化数值错误
不要比较浮点值是否相等来进行循环控制，因为浮点值都是近似值，使用它们可能导致不精确的循环次数和不准确的结果

为了得到更精确的结果，在用for循环计算浮点数的和时应该先加焦晓姝再加较大数，这样可以减少误差

6 关于break和continue
总是可以编写在循环中不使用break和continue的程序，只有在能够简化代码并使程序更容易阅读的情况下，才适合用break和continue

7 设计解决方案时
设计编程解决方案来解决这个问题或其他很多问题的关键之处在于要把问题分解成子问题，然后逐个设计出每个子问题的解决方案。不要一开始就试图开发出一个完整的解决方案。
